# Jogo de Xadrez em Java

Este é um projeto completo de um jogo de xadrez para sistema de terminal (console), desenvolvido inteiramente em **Java**. O objetivo do projeto é aplicar os conceitos de Programação Orientada a Objetos (POO) e lógica de programação avançada.

---

## Funcionalidades

* **Tabuleiro Dinâmico:** Renderização do tabuleiro diretamente no console com suporte a cores para diferenciar as peças.
* **Validação de Movimentos:** Sistema que valida as regras de movimento específicas de cada peça (Peão, Torre, Cavalo, Bispo, Rei e Rainha).
* **Jogadas Especiais:** Implementação de regras clássicas do xadrez, como:
  * *Roque* (Pequeno e Grande)
  * *En Passant*
  * *Promoção de Peão*
* **Lógica de Xeque e Xeque-Mate:** Detecção automática de situações de perigo para o Rei e encerramento da partida.

---

## Tecnologias Utilizadas

* **Linguagem:** Java
* **Conceitos de POO:** Encapsulamento, Herança, Polimorfismo e Tratamento de Exceções.
